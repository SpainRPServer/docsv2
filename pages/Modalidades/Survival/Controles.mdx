# Emotes 
- `B` para abrir la rueda de emotes

# Barcos
- `C` mientras estas moviendo el barco para el control crucero

# Bloques
- `G` para forzar el reenvio de bloques (usar cuando hay bloques invisibles)

# Mapa
- `M` para abrir el mapa

# Roll
- `R` mientras te mueves a la direccion que quieras para rodar en el suelo 

# Skins
- `;` para abrir la configuracion

# Camara
- `ctlr+-` para quitar el renderizado del personaje en primera peresona

# Skills
- `Y` para abrir la interfaz de niveles 

# Ping 
- `J` para abrir la configuracion
- `Boton atras lateral del raton` para pinguear una ubicacion

# Catalejo
- `Z` manten pulsado la z para usar el catalejo en cualquier momento (necesitas tener uno en el inventario)
- `scroll` ajustar el zoom

# Mochila
- `,` para abrir la mochila (necesita estar equipada)

# Chat de voz
- `boton adelante lateral del raton` pulsar para hablar
- `V` configuracion 
- `\` susurrar
- `ctrl+,` para ajustar el volumen de los jugadores
- `I` mutear microfono

# Shaders 
- `ctrl+p` para recargar los Shaders
- `ctrl+k` para quitarlos
